電晶體通道（Channel Boundary）：畫面上下兩條綠色斜線，模擬真實 FinFET 元件中帶有微小「銳角傾斜（acute-angled）」的源極（Source）與汲極（Drain）注入通道。

閘極電場波動（Gate Oxide Gradient）：中央藍色菱形內圈會產生動態呼吸的發光效果，代表這段 DSL 中 IMAGE_kernel().skeleton_set() 鎖定的電晶體閘極反轉層電場。

蒙地卡羅物理變異（Process Variation）：調節「9-Cells Noise」拉桿時，載子（紅點/黃點）會根據晶格缺陷與雜訊大小，在隱藏的 3x3 區塊（9宮格會話鏈）中產生不同程度的布朗擴散，完美符合半導體元件物理模擬的隨機性（Stochastic）。
